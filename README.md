# C Systems Projects

A collection of dsa + low-level systems programming projects written in C.
This repository is built incrementally with daily commits to demonstrate
core data structures, memory management, and system-level thinking.

## Projects
1. Linked List Core
2. Stack & Queue
3. Memory Tracker
4. Log Analyzer
5. Memory Leak Detector
6. Mini Git 

## Environment
- OS: Parrot Security OS
- Language: C (GCC)

## Goal
Build strong fundamentals in dsa + programming through consistent,
documented, hands-on practice.

## Disclaimer
All projects are for educational purposes.
